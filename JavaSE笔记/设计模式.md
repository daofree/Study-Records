##接口--多继承！  interface C extends A,B
    接口是解决单继承局限问题。
    接口=抽象方法+全局常量+内部结构（普通内部类，抽象内部类，内部接口）
    抽象内部类===独立结构，不用实现
    内部接口被 static修饰，就是外部接口，implements A.B
##接口有三大主要功能：
  		· 制订（不同层之间的）操作标准；
  		    电脑和U盘都实现了USB接口，靠接口连接（先有接口标准）
  		· 表示一种（操作的）能力；
  		· 将服务器端的远程方法视图暴露给客户端。

#标准可以连接不同层次的类
    interface USB {	// 操作标准	
    	public void install() ; 
    	public void work() ; 
    } 
    class Computer { 
    	public void plugin(USB usb) { 
    		usb.install() ; 
    		usb.work() ; 
    	} 
    } 
    按照这种方式开发下去的话，不管有多少个USB接口的子类（U盘，打印机...），
    都可以在电脑上使用(plugin)。
